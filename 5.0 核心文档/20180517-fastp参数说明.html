fastp参数及使用效果说明

fastp -i ${dir}/DYDFA-511-2_R1.fq.gz \                                                                #输入文件 rawdata  read1
-I ${dir}/DYDFA-511-2_R2.fq.gz \                                                                         #输入文件 rawdata  read2
-o ${dir}/65014xbycn1A.clean.R1.fastq.gz \                                                         #输出文件 cleandata read1，默认输出为gz格式的压缩文件，压缩等级为2.
-O ${dir}/65014xbycn1A.clean.R2.fastq.gz \                                                        #输出文件 cleandata read2，默认输出为gz格式的压缩文件，压缩等级为2.
-f 5 -F 5 \                                                                                                              #从read1 和read2 起始位置切除5bp，reads起始位置的碱基容易测错，
-3 \                                                                                                                       #开启reads 3’质量剪切
-W 8 -M 25 \                                                                                                        #滑框参数，长度8bp ，滑框内最低平均质量值25
-q 20 -u 25 \                                                                                                        #控制去除质量值低于20的碱基占比大于25%的reads
-n 3 \                                                                                                                    #控制去除含模糊碱基N数量大于3的个数
-l 50 \                                                                                                                    #过滤掉质控后长度低于50bp的reads。
-c \                                                                                                                        #通过PEreads的overlap 关系校正低质量的碱基
-g --poly_g_min_len 6 \                                                                                        #开启polyG tail trimming，ployG长度为6bp，考虑到NextSeq/NovaSeq 数据可能会引入ployG的错误，故设的比较严格
-x \                                                                                                                        #开启polyX tail trimming ，polyX识别长度为默认10bp
-y -Y 20 \                                                                                                              #开启低复杂度过滤，复杂度低于20%的会被去掉，默认不开启，开启后默认复杂度为30%
-j ${dir}/65014xbycn1A.json \                                                                              #json格式质控统计结果文件，默认为fastp.json
-h ${dir}/65014xbycn1A.html \                                                                            #html格式质控统计结果文件，默认为fastp.json
--adapter_sequence AGATCGGAAGAGCACACGTCTGAACTCCAGTCAC \           #额外指定的接头序列，read1 的接头序列
--adapter_sequence_r2 AGATCGGAAGAGCGTCGTGTAGGGAAAGAGTGT          #额外指定的接头序列，read2 的接头序列
